# Financial-Stock-Assistant-Chatbot
This project is a GPT-powered financial chatbot designed to provide real-time stock market insights through natural language interaction. Users can query stock prices, trends, and technical indicators, making financial analysis more accessible and interactive.

The system integrates live and historical stock data to compute indicators such as RSI, MACD, and Moving Averages, presenting results through clear visualizations. An intuitive Streamlit-based interface allows users to explore data, view charts, and receive AI-generated explanations in real time.

This project demonstrates the practical application of AI, data analysis, and visualization in the FinTech domain, offering a scalable foundation for intelligent investment analysis tools.
